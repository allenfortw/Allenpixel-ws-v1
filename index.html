如何加入

只需簡單幾步，立即加入我們的伺服器，展開你的起床戰爭旅程！

1

準備 Minecraft Java 版

確保你擁有最新版本的 Minecraft Java 版本遊戲。我們支援 1.8 - 1.20.2 版本。
